## 🚀 使用方式

### 1. 访问页面
- 登录系统后，点击左侧菜单 "产品管理" → "代理记账服务"
- 直接访问：http://localhost:5174/bookkeeping-packages

### 2. 管理套餐
1. 点击左侧 "新增套餐" 按钮
2. 填写套餐信息（名称、描述、价格等）
3. 保存后套餐出现在左侧列表中

### 3. 管理服务
1. 点击右上角 "新增代账服务" 按钮
2. 填写服务信息（名称、说明、价格、属性等）
3. 保存后服务出现在右侧表格中

### 4. 配置套餐服务
1. 在右侧表格中找到对应服务行
2. 在套餐列中切换开关
3. 开关状态实时保存
